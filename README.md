# Wordle Arena

Wordle Arena is a Python-based GUI word-guessing game inspired by Wordle.
Players guess 4, 5, or 6-letter English words within six attempts and receive
color-coded feedback.

## Features
- Multiple word lengths (4, 5, 6)
- Themes and animations
- Dynamic on-screen keyboard
- Optional timer with selectable durations
- Score tracking
- Displays word meaning at the end of each game

## Requirements
- Python 3.x
- requests library

## How to Run
1. Clone or download the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the game:
   python main.py
